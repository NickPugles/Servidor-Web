Aula-10 (Array):
- Array: é uma variável que guarda vários valores, possui diversos espaços disponíveis para colocar uma informação;
-A...(): dentro das chaves deve possuir o identificador, qual informação deve aparecer;
-A chave deve sempre estar presente;
-O <br> deve ser usado na mesma linha após o ponto final e com aspas, exemplo: echo $estado ['1']. '<br>';
-[], (), {}: é uma chave, onde ela associa o que deve ser inserido;
-array associativa, possui =>;
-array com muitas informações, porém cada uma possui um id;
-array_unshift: acrescenta elementos na frente/ início do array;
-foreach: ele é responsável por transferir o valor da primeira variável para a próxima. Portanto, ele percorre o caminho e vai apagando o que já foi percorrido;
-array_push: acrescenta no final de array;
-array_shift: remove do início;
-array_key: retorna somente as chaves;
-array_values: retorna somente os valores;
-array_reverse(): inverte os elementos, exemplo, o que está em cima vai para baixo e o que está em baixo para cima;
-array_short: coloca de maior para o menor ou em ordem alfabética;
-array_flip: inverte as funções dos valores com as chaves. Obs: ocorre em casos específicos;
-array_marge: junta as array e coloca em uma única;
